    JacquardProtocol.ImuSample.Builder

    public static final class JacquardProtocol.ImuSample.Builder extends implements ImuSampleOrBuilder 

     An ImuSample represents a discrete sample from a Jacquard IMU sensor
 (inertial measurement unit). Note, the int32 IMU values below are output
 as int16 values from the hardware.  Protobufs do not explicitly support int16
 values, however, they do use variable-width integers on the wire, more
 details on this can be found here:
 https://developers.google.com/protocol-buffers/docs/encoding#varints
 TODO(b/150927507): correct the names of datapoints obtained from the gyro.

    hasUjtSysTick

    public boolean hasUjtSysTick() 

     UJT (Universal Jacquard Tag) system tick counter, this is a relative
 counter that represents system ticks of the UJT's microprocessor. The
 processor runs at 32K, therefore this value will increase 32K times a
 second. The hardware counter will reset to zero upon a hardware reset or
 when the tick counter overflows.

    optional uint32 ujt_sys_tick = 1;
